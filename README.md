# The-SaxplayerTM
This is the repo for my Saxplayer! It is my own sample demo project for my tutorial to soon be posted on stasis! It is a simple mp3 player/streamer using a seeed, a headphone jack, an oled, and 3 buttons/keyswitches.

**DESCRIPTION OF THE INTERNALS**
My Saxplayer is a bare bones PCB that houses a XIAO-RP2040-DIP for the microcontroller, 3 MX-Style switches for navigation, a rotary encoder switch for adjusting the volume with a knob, and a audio jack for headphones.
